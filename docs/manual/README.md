# MindScript Documentation

This is the documentation for the programming language MindScript.

It is built using [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/). Please refer to the documentation for more details.

**Important**: For development, you must install the package:
```
pip install mkdocs-material
```

You might have to install additional packages, depending on the features of the site.

Then, run the server with
```
mkdocs serve
```
